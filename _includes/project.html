Baby-Namr

  • Baby-Namr helps expectant parents research potential baby names by providing historical information on name popularity in the United States.
  • Baby-Namr's backend is an API built in Rails which pulls data from a PostgreSQL database and serves JSON to a React frontend.
  • Data was parsed from CSV files provided by the Social Security Administration.
